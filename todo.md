# Project Tasks

- [ ] **Register Application**: Go to [TickTick Developer Center](https://developer.ticktick.com/manage) and register a new app to get `CLIENT_ID` and `CLIENT_SECRET`.
    - Redirect URL: `http://127.0.0.1:5000/callback`
- [ ] **Environment Setup**: Create `.env` file with the credentials.
- [ ] **Install Dependencies**:
    - Create virtual env: `python -m venv ticktick`
    - Activate it: `source ticktick/bin/activate` (Linux/Mac) or `ticktick\Scripts\activate` (Windows)
    - Install: `pip install -r requirements.txt`
- [ ] **Run App**: Start the server with `python app.py`.
